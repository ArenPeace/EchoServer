## 代理
**正向代理/反向代理**  
正向代理，即翻墙/隐藏客户端访问路径/堡垒机安全访问/VPN专线访问。反向代理，保护或隐藏网络服务或资源/加密和SSL加速/负载均衡/缓存CDN/压缩/减速上传/安全/外网发布。区分正向代理还是反向代理的关键不是是否在同一局域网或防火墙前后，而是代理的目的或为谁服务。
