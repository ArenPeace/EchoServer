### 交收（清算/结算）
T+n滚动交收 VS 实时交收

清偿收付双方债券债务关系的过程及方法。同行账户资金往来直接结算，不同行之间账户资金往来的，需要先清分再结算。
+ 清分  Clearing对交易数据依据机构和交易类型进行分类汇总，并计算结算金额的过程。
+ 清算 Settlement指根据清分结果对交易数据进行净额轧差和提交并完成资金划拨的全过程。
+ 结算  Settlement of Accounts指完成客户账户间资金划拨的全过程。
